# O Essencialista

## Sobre o Projeto

O Essencialista é um blog/site cristão dedicado a compartilhar reflexões e ensinamentos bíblicos, focando no que é essencial para a fé cristã. O site tem como objetivo ajudar os leitores a descobrirem e se aprofundarem nas verdades fundamentais das Escrituras.

> "Tudo que foi escrito, para nosso ensino foi escrito. A fim de que pela consolação das Escrituras tenhamos esperança" - Romanos 15:4

## Características

- Design responsivo usando Tailwind CSS
- Coleção de artigos sobre temas cristãos
- Interface limpa e intuitiva
- Otimizado para diferentes dispositivos

## Tecnologias Utilizadas

- HTML5
- CSS3
- Tailwind CSS
- Font Awesome
- JavaScript
